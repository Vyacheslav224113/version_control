# Посказки по командной строке

Команда смены директории 
```sh
cd c:\folder_name
```

Команда отображения текущей директории
```sh
pwd
```

Листинг текущей директории
Windows:
```sh
dir
```
Linus, MacOS
```sh
ls
```

Удаление файла 
WIn: 
```sh
del<Filename>
```
в Linux, MacOS 
```sh
rm<Filename>
```
